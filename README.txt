JPG -> JPEG Smart Renamer Portable

Run:
After publishing, double-click Start-Renamer.bat or JPG_to_JPEG_Portable.exe.

Build:
dotnet publish -c Release

Single-file output:
bin\Release\net8.0-windows\win-x64\publish\JPG_to_JPEG_Portable.exe

Features:
- Browse folder
- Rename .jpg to .jpeg
- Duplicate-safe numbering
- Preview mode
- Recursive option
- Log file
